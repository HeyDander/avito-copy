# List.am Next

Современный classified marketplace в духе Avito, но с собственной визуальной системой, расширенными функциями и архитектурой под Netlify.

## Что уже есть

- современный фронтенд на `React + Vite`
- serverless backend на `Netlify Functions`
- лента объявлений с фильтрами и поиском
- избранное
- сообщения между продавцом и покупателем
- кабинет продавца с публикацией новых объявлений
- аналитический блок по категориям

## Как запускать локально

```bash
npm install
npm run dev
```

Локальный сервер будет доступен по адресу `http://localhost:8888`.

## Основные команды

```bash
npm run dev
npm run dev:ui
npm run lint
npm run build
```

## Структура

- `src/` - клиентское приложение
- `netlify/functions/` - backend API
- `netlify/functions/_lib/` - общее состояние и seed-данные
- `netlify.toml` - конфигурация Netlify и API redirects

## Важное замечание по backend

Сейчас backend сделан как демо-ready слой на `Netlify Functions` с состоянием в памяти процесса. Для локальной разработки и демонстрации этого достаточно, но для полноценного production-деплоя на Netlify нужно подключить постоянное хранилище:

- Supabase
- Neon/Postgres
- PlanetScale
- Netlify Blobs / внешнюю БД

Если захотите, следующим сообщением я могу сразу сделать второй этап: авторизацию, реальную базу данных, загрузку фото, личные кабинеты и админку.
